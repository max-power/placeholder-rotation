Placeholder
Rotation

new PlaceholderRotation(document.getElementById('search'), {
  terms: ['Ruby', 'Python', 'Scala', 'Go', 'Smalltalk'],
  template: 'Are you looking for {term}?',
  delay: 250
})

or use HTML5 Data Attributes:

<input id="s" type="text" data-terms="Javascript,Madness" data-delay="1000">
<script>
  new PlaceholderRotation(document.getElementById('s'))   
</script>

You can use the following attributes:

terms
Array
String (seperated with comma) for data-terms
default: ["something"]
template
String (must include {term})
default: "Search for {term}"
delay
Integer (in milliseconds)
default: 1000
index
Integer, Index of the first element to show
default: 0
